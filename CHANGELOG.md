### Added

- Initial `clowk-terminal` builder: one canonical palette → many terminal formats.
- **Clowk Night** palette, imported from `clowk-vscode-theme`.
- Generators: Ghostty (also covers cmux), iTerm2, macOS Terminal.app, Alacritty,
  kitty, WezTerm, Windows Terminal.
- `bun run import` / `build` / `verify` pipeline.
- CI and tagged-release workflows (GitHub Actions).
